# Tic-Tac-Toe Game
### Description
A simple and interactive TIC-TAC-TOE game built using MERN stack. It include user authentication system implemented using JWT (JSON Web Token)

 The application starts with user authentication using JWT, allowing access only after login or sign-up. Users can verify their email addresses to enable password recovery through OTP in case of a forgotten password.

Once logged in, players can enjoy the Tic-Tac-Toe game in two-player mode. The game declares the winner upon completion, providing an engaging and seamless experience.

### Features
- Secure user authentication with JWT.
- Email verification for password recovery.
- Two-player Tic-Tac-Toe gameplay.
- Highlights the winner or declares a draw.

### Tech Stack
- **React JS** for frontend layout
- **Node JS** for backend development
- **Tailwind CSS** for styling
- **Mongo DB** for data storing
- **Express** and **Socket** for server connection
